常见问题 FAQ

Q: 双击 Install.command 没有反应？ Q: Double-clicking Install.command does nothing?
请确保已正确解压 ZIP 文件。如果问题仍然存在，可以右键点击 Install.command，选择「打开方式」→「终端」。 Make sure you've properly unzipped the ZIP file. If the issue persists, right-click Install.command and select "Open With" → "Terminal".
Q: 显示"应用已损坏"怎么办？ Q: App shows "damaged" error?
这是 macOS 安全机制。安装脚本会自动处理。如果仍有问题，可运行诊断工具 diagnose.sh 检查。 This is macOS security. The install script handles it automatically. If issues persist, run diagnose.sh to check.
Q: 重启后需要重新输入密码？ Q: Need to re-enter password after restart?
v2.0+ 已修复此问题。安装时只需输入一次密码，之后永久免密。 v2.0+ fixed this. You only need to enter password once during installation.
Q: 如何卸载？ Q: How to uninstall?
在解压目录中双击 uninstall.sh 文件即可卸载。 Double-click uninstall.sh in the extracted folder to uninstall.
Q: 如何检查安装状态？ Q: How to check installation status?
在解压目录中双击 diagnose.sh 文件，会显示详细的安装状态。 Double-click diagnose.sh in the extracted folder to see detailed installation status.
