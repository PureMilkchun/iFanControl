Double-click the ZIP to extract it to Downloads folder

No paths to type, no commands to remember

打开「终端」应用（按 ⌘ + 空格 搜索"终端"），然后：

Open "Terminal" app (press ⌘ + Space and search "Terminal"), then:

  1. 找到解压后的文件夹中的 install.sh 文件
  2. Find the install.sh file in the extracted folder
  3. 将 install.sh 拖入终端窗口
  4. Drag install.sh into the Terminal window
  5. 按回车执行
  6. Press Enter to run
💡 密码提示： 安装过程中会要求输入密码。输入时屏幕上不会显示任何字符（包括星号），这是正常的！直接输入密码后按回车即可。 💡 Password tip: During installation, you'll be asked for your password. Nothing will appear on screen when typing (not even asterisks) - this is normal! Just type your password and press Enter.
⚠ 请不要双击 Install.command： 从网络下载后，`.command` 文件可能被 macOS 持续拦截。请始终使用“拖拽 install.sh 到终端”的方式安装。 ⚠ Avoid double-clicking Install.command: After downloading from the internet, `.command` files may be repeatedly blocked by macOS. Always install by dragging `install.sh` into Terminal.

常见问题 FAQ

Q: 拖入终端后没有反应？ Q: Nothing happens after dragging to Terminal?
请确保已正确解压 ZIP 文件，并且拖入的是 install.sh 文件。拖入后需要按回车键执行。 Make sure you've properly unzipped the ZIP file and dragged the install.sh file. You need to press Enter after dragging.
Q: 显示"应用已损坏"怎么办？ Q: App shows "damaged" error?
这是 macOS 安全机制。安装脚本会自动处理。如果仍有问题，可运行诊断工具 diagnose.sh 检查。 This is macOS security. The install script handles it automatically. If issues persist, run diagnose.sh to check.
Q: 为什么双击 Install.command 一直被拦截？ Q: Why is Install.command repeatedly blocked?
这是下载来源触发的 Gatekeeper 行为。请改用推荐路径：打开终端，拖入 install.sh，按回车执行。 This is expected Gatekeeper behavior for internet downloads. Use the recommended path: open Terminal, drag install.sh in, and press Enter.
Q: 重启后需要重新输入密码？ Q: Need to re-enter password after restart?
当前版本已修复此问题。安装时只需输入一次密码，之后即可免密运行。 Current versions already fix this. You only need to enter your password once during installation.
Q: 如何卸载？ Q: How to uninstall?
打开终端 → 将 uninstall.sh 拖入终端 → 按回车 Open Terminal → Drag uninstall.sh into Terminal → Press Enter
Q: 如何检查安装状态？ Q: How to check installation status?
打开终端 → 将 diagnose.sh 拖入终端 → 按回车 Open Terminal → Drag diagnose.sh into Terminal → Press Enter
