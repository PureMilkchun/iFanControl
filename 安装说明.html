常见问题 FAQ

Q: 显示"应用已损坏"怎么办？ Q: App shows "damaged" error?
这是 macOS 安全机制。安装脚本会自动处理，如果仍有问题，可在终端运行： This is macOS security. The install script handles it automatically. If still having issues, run in Terminal:
sudo xattr -cr /Applications/MacFanControl.app
Q: 重启后需要重新输入密码？ Q: Need to re-enter password after restart?
v2.0 已修复此问题。安装时只需输入一次密码，之后永久免密。 v2.0 fixed this. You only need to enter password once during installation.
Q: 如何卸载？ Q: How to uninstall?
运行解压目录中的 uninstall.sh，或在终端执行： Run uninstall.sh from the extracted folder, or execute in Terminal:
cd ~/Downloads/MacFanControl-2.0 && ./uninstall.sh
